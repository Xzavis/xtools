## üöÄ CORE_MODULES

### 1. **SYSTEM_OPERATIONS**
*   **Split Engine**: Partition large binary files (weights/datasets) into manageable segments.
*   **Merge Logic**: Reconstruct segmented manifests back into their original binary form.
*   **JSON Extractor**: Instant header extraction for `.safetensors` and CSV conversion.

### 2. **AI_RESEARCH_SUITE**
*   **Model Hub**: Seamless integration with HuggingFace Hub for high-speed batch retrieval.
*   **Neural Inspector**: Deep-dive into `.safetensors` geometry and layer mapping without memory overhead.
*   **Dataset Sanitizer**: Analyze and clean CSV/JSON datasets (deduplication, null detection).
*   **RAG Architect**: Semantic text partitioning for Retrieval-Augmented Generation pipelines.

### 3. **COMPUTE_OPS**
*   **VRAM Calculator**: Precision hardware projection based on model parameters and quantization bit-depth.

### 4. **UTILITY_TOOLBOX**
*   **Universal Converter**: Transcoding visual assets, structured data, and documents.
*   **Snippet Lab**: Persistent, syntax-highlighted markdown storage for code and prompts.

## üõ†Ô∏è INSTALLATION_GUIDE

### Prerequisites
*   Python 3.9+
*   Fast Internet Connection (for Model Hub)

### Setup Sequence
1. **Clone Repository**
   ```bash
   git clone https://github.com/yourusername/xtools.git
   cd xtools
   ```

2. **Environment Configuration**
   ```bash
   python -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```

3. **Initialize Core**
   ```bash
   python app.py
   ```
   Access interface at `http://127.0.0.1:5000`
